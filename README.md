# Motion Detector using OpenCV

A beginner computer vision project built using Python, OpenCV, and NumPy.

## Files

### part1.py

Basic motion detector using:

* Frame differencing
* Grayscale conversion
* Thresholding
* Pixel change counting

### part2.py

Enhanced motion detector with:

* Motion detection
* Image saving
* Alarm sound notification
* Visual rectangle alert

## Features

* Captures live webcam feed
* Compares consecutive frames
* Detects motion using frame differencing
* Uses thresholding to reduce noise
* Displays motion detection in real time
* Saves an image when motion is detected
* Plays an alarm sound when motion is detected

## Technologies Used

* Python
* OpenCV
* NumPy
* Winsound
